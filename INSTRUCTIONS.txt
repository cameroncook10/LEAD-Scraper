================================================================================
                        REQUIRED API KEYS (3 NEEDED)
================================================================================

Before running, you need:

1. SUPABASE_URL
   - Create free account at https://supabase.com
   - Create new project
   - Copy Project URL from Settings > API

2. SUPABASE_ANON_KEY
   - In same Supabase project
   - Settings > API > anon public key

3. ANTHROPIC_API_KEY
   - Create free account at https://console.anthropic.com
   - Go to Account > API Keys
   - Create new API key

Then:
1. Copy .env.example to .env
2. Paste these 3 keys into .env
3. Also run the SQL schema in Supabase (see QUICK_START.md)
4. Run: npm run dev
