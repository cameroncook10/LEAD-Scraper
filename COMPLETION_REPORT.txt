================================================================================
                          API ENDPOINTS
================================================================================

Scrape Jobs:
- POST /api/scrape/start - Create scrape job
- GET /api/scrape/status/:jobId - Get job status
- GET /api/scrape/jobs - List all jobs

Leads:
- GET /api/leads - Get leads (filterable, paginated)
- GET /api/leads/:id - Get single lead
- GET /api/leads/stats/summary - Get statistics
- POST /api/leads/export - Export to CSV
- DELETE /api/leads/:id - Delete lead

Jobs:
- GET /api/jobs - List jobs
- GET /api/jobs/:jobId - Get job with logs

All endpoints return JSON with CORS enabled
