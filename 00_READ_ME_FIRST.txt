================================================================================
                        YOU NEED 3 API KEYS
================================================================================

Get these first (all have free tiers):

1. SUPABASE (free database)
   → https://supabase.com
   → Create project
   → Settings > API > Copy URL and anon key

2. ANTHROPIC (free AI API)
   → https://console.anthropic.com
   → Create API key
   → Copy the key

Then:
1. Edit .env file
2. Paste the 3 keys
3. Create database schema
4. Run npm run dev
